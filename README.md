# crud-app-27

## Usage

Run

```bash
docker-compose up
```

Run (background process)

```bash
docker-compose up -d
```

Remove

```bash
docker-compose down
```

## Docker Containers

- Frontend:
  - `web-app-prd` for production build
  - `web-app-dev` for hot reloadable dev build
- Backend:
  - PHP
  - Apache
- Database:
  - MySQL
